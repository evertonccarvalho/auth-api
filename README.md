## Description

## Instalation and running the app locally

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Running with docker

To run the application with docker, first you need to deactivate postgresql if it's activated, then run the following commands:

# to run docker-compose

```bash
$ docker-compose up -d
```

You can access the API at localhost:3000 and the documentation at localhost:3000/api.
